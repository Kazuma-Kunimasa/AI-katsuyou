## 2. 生成AIとchatGPTの仕組みについて

### 生成AIについて

生成AI（Generative AI）とは、人工知能の一分野であり、新しいデータを生成する能力を持つモデルを指す。

chatGPTは⽣成AIを活⽤したサービスです！

> *⽣成AI（ジェネレーティブAI）とは？ChatGPTとの違いや仕組み‧種類‧活⽤事例*

### ChatGPTについて

ChatGPTとは？（https://chatgpt.com/）

OpenAI社が開発する、チャットボット型のAI。
ユーザーが質問やリクエストをすると、それに対して回答や提案を⽣成してくれる。
ウェブ版、iOS‧iPad版、Android版、MacOS版で利⽤可能。

#### ChatGPTができること

- ⽂章作成
- データ分析
- ロールプレイ
- お悩み相談
- プログラミング
- プログラム実⾏
- 画像認識
- 画像⽣成
- 調査（webブラウジング）
- 語学学習
- その他多数

### ChatGPTの裏側

ChatGPTの裏側の仕組み：
膨⼤なテキストデータから「次に来る⽂字」を予測し、それをどんどん繋げていくことで⽂章を⽣成。

AIが様々な出⼒を試して最も良い出⼒で再学習し、⼤量に単語を学習したら⼈間が好む出⼒を⽣成できるように。

#### まとめ

ChatGPTは次に続く可能性が確率上、⾼い⾔葉を予測しているだけ。

1. ⼀般的なことしか⾔わない
2. 学習ソースが間違っていればChatGPTも間違える

## 3. 初期設定と基本操作について

## 4. chatGPTの最新モデルgpt4oについて

### 他モデルとの競争

⽣成AIは⽇々モデルが更新されており、サプライヤー間で性能の競争が⾏われている。
現状最強モデルはAnthropic社のclaude3.5Sonnetと⾔われている（7/1現在）。

> *AnthropicがClaude 3.5 Sonnetを突如発表、ベンチマーク結果はGPT-4oに匹敵（livedoornews）*

### GPT-4oとは

OpenAI社が開発する、最新のAIモデル。(2024年5⽉公開)
従来のChatGPTはテキストのみのやり取りだったが、GPT-4oは、⾳声、画像、テキストをリアルタイムに理解し、テキスト、⾳声、画像で出⼒することができる。
オムニ（Omni）は「すべての」という意味。

#### モデル比較

| モデル | ユーザーからの⼊⼒ | AIからの出⼒ |
|--------|---------------------|--------------|
| GPT-4  | テキスト            | テキスト     |
| GPT-4o | テキスト 画像 ⾳声 映像 | テキスト 画像 ⾳声※現在デモのみ 映像※現在デモのみ |

#### GPT-4oの凄いところ

- マルチモーダル対応: テキスト、⾳声、画像の組み合わせを⼊⼒として受け取り、同様に多様な形式の出⼒を⽣成可能。
- ⾼速応答: ⾳声⼊⼒に対して232ミリ秒、平均320ミリ秒で応答し、⼈間の会話とほぼ同等のスピード。
- 多⾔語対応: GPT-4 Turbo（既存の上位モデル）と同等の英語およびコード処理能⼒を持ち、⾮英語テキストでも⼤幅に改善。
- コスト効率: APIではGPT-4 Turboよりも2倍⾼速で、価格は半分、レート制限は5倍に。

> *【📍完全版】世界が震撼したOpenAI最新アップデートまとめ《GPT-4o、デスクトップアプリ、⾳声会話》*

- 【⾳声出⼒】⽇本語（関⻄弁）やラップなどもできる
- 【⾳声⼊⼒】だらだら喋っても理解してくれる（議事録作成も可能）
- 【画像⼊⼒&出⼒】ユーザーが提供した画像を基に新たな画像を作れる

## 6. 効果的なAIへの指⽰「プロンプト」の書き⽅

### プロンプトとは

- AIに対してどのようなタスクを実⾏してほしいかを記載した指⽰⽂のこと。
- プロンプト次第で出⼒が⼤きく変わるので、とても重要。
- ⽣成AIの活⽤⽅法を学ぶ＝プロンプトを学ぶといっても過⾔ではない。
- より効果的な出⼒を得るためのプロンプトデザイン⼿法をプロンプトエンジニアリングといい、近年体系的にまとめられ始めている。

### 効果的なプロンプトを書くためのコツ8選

1. 単語にこだわる
2. AIが理解しやすい形式で書く（マークダウン形式）
3. 役割を与える
4. 明確な指⽰を書く
5. ⽂脈情報の提供
6. タスクを段階的に分ける
7. 考える時間を与える
8. 実例を与える

#### 1. 単語にこだわる

例：「紫式部、ラブレター書いて」

- 「紫式部」という単語が⼊るだけで全然違う出⼒になる
- プロンプトで⾔語空間を絞り込むことで特定の回答が得られる
- 専⾨的なタスクを実⾏させたい場合は、事前に重要な専⾨⽤語をプロンプトに含むことが、より⾼度な回答が得られる

おすすめのプロンプト例：
「○○における専⾨⽤語を⼀覧で教えて」

| 用語 | 意味 |
|------|------|
| ROI | 投資利益率 |
| CPA | 獲得単価 |
| CPC | クリック単価 |
| CTR | クリック率 |
| SEO | 検索エンジン最適化 |
| SEM | 検索エンジンマーケティング |
| PPC | クリック課⾦型広告 |

#### 2. マークダウン記法について

- ChatGPTに対しても「何をしてほしいのか」を明確に⽰してあげる必要がある
- 文章の構造や関連性がわかりやすいプロンプトの作成にはMarkdown記法が便利

##### Markdown記法の例

| 機能 | 記法 | 説明 |
|------|------|------|
| 見出し | # | h1見出し、##はh2見出しなど階層化 |
| リスト | - または * | 箇条書き |
| 番号付きリスト | 1. 2. | 頭に数字が付くリスト |
| コードブロック | ``` | コードやコマンドを囲む |
| テーブル | | | 表形式 |

##### 良いプロンプト例

```
# 仕様
- Markdown記法を使⽤すること
- 記事全体の⽂字数は500字〜1000⽂字とすること
- 記事の最後に以下情報を⼊れること
```
※この記事はChatGPTを使⽤して作成されています。
執筆⽇：YYYY/MM/DD
執筆者：John Doe
```

# 対象読者
記事の対象読者は以下の要素を含む⼈物とする。
- ChatGPT未経験者
- ITエンジニア

# 記事タイトル
ChatGPT⼊⾨ガイド

# 記事内容
前述の# 対象読者を考慮し、以下内容で記事を作成すること。
1. ChatGPTについて解説
2. ChatGPTを使⽤する上でのメリットとデメリットについて
3. ChatGPTの使⽤例
4. GPT-3.5とGPT-4.0の違いについて以下の表形式に沿って説明してください。
|特徴|GPT-3.5|GPT-4.0|
|料⾦体系|||
|性能|||
```

#### 3. 役割を与える

- 事前に役割を明記してあげると、それに従って回答を作ってくれる
- 例：「あなたは経験豊富な百戦錬磨のP&Gのマーケターです」

#### 4. 明確な指⽰を書く

- 曖昧な指⽰では曖昧な回答しか返ってこない
- 例：「⽝の画像を作成して」→「茶⾊の柴⽝が公園で⾛っている画像を作成して」

#### 5. ⽂脈情報の提供

- タスクの前提となる⽂脈を与えるとそれを汲み取って回答してくれる
- 例：「新製品のマーケティング戦略を提案してください」

#### 6. タスクを段階的に分ける

- タスクの処理フローを⽰すとそれを守ってくれる
- 例：「新しいプロジェクトの管理⽅法を教えてください」→「以下の順番でタスクを実⾏してください」

#### 7. 考える時間を与える

- 「ステップbyステップ」を追加すると考える時間が⽣まれ、回答の精度が⾼まることがある

#### 8. 実例を与える

- 具体例を与えると、求めている回答に近い出⼒が出せる

### 検索特化型AIアプリ「perplexity」

- 検索タスクに特化したAIアプリケーション
- 無料でも使える
- 対話型で質問をチャット欄に投げ込むだけ
- Focusから検索結果の引⽤元に制限をかけられる（例：Academicは論⽂からのみ回答）
- 引⽤元が検索結果と同時に表⽰されるため、ソースをすぐ辿れる
- 無料版と有料版(Perplexity Pro)の主な違いもあり
